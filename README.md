# WhatsApp Bot Baileys

Este é um bot de WhatsApp usando a biblioteca Baileys para Node.js. Ele responde ao comando de grupo:

```
!-- sou hetero?
```

Com uma chance de 20% de responder "Sim" e 80% de responder "Não".

## Como usar

1. Instale as dependências:
   ```
   npm install
   ```
2. Inicie o bot:
   ```
   npm start
   ```
3. Escaneie o QR code com o WhatsApp para autenticar.
4. Envie o comando em um grupo para testar.

## Requisitos
- Node.js 18+
- WhatsApp em seu celular

## Observações
- O bot só responde em grupos.
- O comando deve ser exatamente: `!-- sou hetero?`
